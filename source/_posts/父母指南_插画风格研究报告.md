# 「父母指南」小程序插画风格类型研究报告

> 为28道测评题配图的场景插画风格选型指南，含MVP推荐方案与AI生成提示词模板。

---

## 一、微信小程序图片约束（先了解边界）

| 约束项 | 限制 | 对插画的影响 |
|--------|------|-------------|
| **小程序包大小** | 2MB（整个小程序代码+资源） | 28张图不能全部打包，需CDN外链或按需加载 |
| **单张图片建议** | ≤200KB | 复杂细节（水彩晕染、3D渲染）难以压缩到这个大小 |
| **推荐格式** | WebP > PNG > JPG | 透明背景用PNG，照片质感用JPG，通用用WebP |
| **推荐宽度** | 750px（iPhone标准） | 所有图片统一宽度，高度自适应 |
| **加载体验** | 弱网环境下3秒内可见 | 线条简单、色块少的风格加载更快 |

**核心结论**：风格必须「轻量」——色块少、线条简、层次少，否则28张图加载体验会崩。

## 二、10种插画风格详细分析

### 1. 扁平插画（Flat Illustration）

| 维度 | 评价 |
|------|------|
| **特点** | 纯色色块、无渐变、无描边、几何感强、人物圆润 |
| **视觉情绪** | 现代、干净、友好、不压迫 |
| **适合年龄段** | 全年龄段通用 |
| **MVP可实现性** | ⭐⭐⭐⭐⭐ **最高** |
| **优点** | 文件极小（50-100KB）、加载快、风格统一易控制、矢量可无限缩放 |
| **缺点** | 情感细腻度有限、容易显得"模板感"、辨识度一般 |
| **情绪一致性** | 温暖可控——暖橙+暖白+炭灰配色体系直接套用 |

**典型参考**：Headspace冥想App、Duolingo、支付宝生活号插画

---

### 2. 手绘卡通（Hand-drawn Cartoon）

| 维度 | 评价 |
|------|------|
| **特点** | 圆润线条、可爱夸张、略带不规则、亲和力极强 |
| **视觉情绪** | 童趣、柔软、治愈、降低焦虑 |
| **适合年龄段** | 0-6岁题目最佳，7-12岁可用 |
| **MVP可实现性** | ⭐⭐⭐⭐ 高 |
| **优点** | 情绪价值高、用户容易产生好感、裂变分享时更吸引人 |
| **缺点** | 线条不规则导致文件稍大（100-150KB）、风格统一需要同一画师 |
| **情绪一致性** | 非常温暖——手绘的不完美本身就是"人味" |

**典型参考**：Line Friends、熊本熊绘本、小红书育儿类封面

---

### 3. 线条插画（Line Art / 单色线稿）

| 维度 | 评价 |
|------|------|
| **特点** | 单色/双色、仅用线条勾勒、极简、大量留白 |
| **视觉情绪** | 克制、安静、不打扰、高级感 |
| **适合年龄段** | 7-18岁题目（尤其是13-18岁敏感场景） |
| **MVP可实现性** | ⭐⭐⭐⭐⭐ **最高** |
| **优点** | 文件最小（30-80KB）、加载最快、高级感强、不抢答题注意力 |
| **缺点** | 0-3岁场景可能显得太冷、情绪表达有限 |
| **情绪一致性** | 中性温暖——线条+暖色点缀即可软化 |

**典型参考**：Notion官网插画、The New Yorker漫画风格

---

### 4. 水彩/水墨风（Watercolor）

| 维度 | 评价 |
|------|------|
| **特点** | 柔和晕染、边缘模糊、文艺感、情绪层次丰富 |
| **视觉情绪** | 温柔、治愈、有故事感、适合情绪类场景 |
| **适合年龄段** | 情绪冲突场景（孩子哭泣、家长疲惫、深夜崩溃） |
| **MVP可实现性** | ⭐⭐ 低 |
| **优点** | 情绪表达最强、用户记忆点深、文艺父母群体喜欢 |
| **缺点** | 文件巨大（200-500KB，难压缩）、加载慢、风格统一难度大 |
| **情绪一致性** | 非常温暖——但技术实现不友好 |

**典型参考**：Instagram育儿博主封面、日系绘本

---

### 5. 绘本风格（Picture Book）

| 维度 | 评价 |
|------|------|
| **特点** | 类似儿童绘本、有叙事感、场景完整、人物有表情动作 |
| **视觉情绪** | 沉浸感强、像在翻一本故事书 |
| **适合年龄段** | 全年龄段 |
| **MVP可实现性** | ⭐⭐⭐ 中 |
| **优点** | 故事感最强、用户代入感深、品牌辨识度高 |
| **缺点** | 单图信息量大导致文件偏大（150-250KB）、需要系列化设计能力 |
| **情绪一致性** | 温暖——绘本天生带治愈属性 |

**典型参考**：《猜猜我有多爱你》风格、Eric Carle拼贴风格

---

### 6. 2.5D / 等距视角（Isometric）

| 维度 | 评价 |
|------|------|
| **特点** | 立体但不完全3D、等距视角、建筑/场景感强 |
| **视觉情绪** | 现代、结构化、信息清晰 |
| **适合年龄段** | 7-12岁（写作业、考试、游戏场景） |
| **MVP可实现性** | ⭐⭐⭐ 中 |
| **优点** | 场景信息量大、适合展示"空间关系"（如客厅打架场景） |
| **缺点** | 人物情绪表达弱、文件中等（100-180KB）、风格偏冷 |
| **情绪一致性** | 中性——需要暖色调和才能变暖 |

**典型参考**：Google Material Design插图、Notion模板封面

---

### 7. 黏土风 / 立体插画风（Claymation-style）

| 维度 | 评价 |
|------|------|
| **特点** | 像黏土捏的、圆鼓鼓、质感柔软、立体 |
| **视觉情绪** | 童趣、柔软、忍不住想捏、治愈 |
| **适合年龄段** | 0-6岁 |
| **MVP可实现性** | ⭐⭐ 低 |
| **优点** | 辨识度极高、情绪价值满分、裂变时很吸睛 |
| **缺点** | 文件大（200KB+）、AI生成不稳定、需要后期大量修图 |
| **情绪一致性** | 非常温暖——黏土质感自带亲和力 |

**典型参考**：黏土动画《小羊肖恩》、小红书黏土滤镜内容

---

### 8. 极简 Icon 风（Minimal Icon）

| 维度 | 评价 |
|------|------|
| **特点** | 符号化、高度抽象、无人物表情、仅用图形暗示场景 |
| **视觉情绪** | 冷静、理性、不评判 |
| **适合年龄段** | 全年龄段（尤其是怕"被定义"的用户） |
| **MVP可实现性** | ⭐⭐⭐⭐⭐ **最高** |
| **优点** | 文件最小（20-50KB）、加载最快、完全不judgemental、SVG可无限缩放 |
| **缺点** | 情绪传达极弱、代入感差、可能显得"太冷" |
| **情绪一致性** | 中性——需要配色变暖 |

**典型参考**：Apple Health图标、Nike Run Club、极简App引导页

---

### 9. 国潮 / 新中式（Neo-Chinese）

| 维度 | 评价 |
|------|------|
| **特点** | 传统水墨/工笔元素+现代设计、东方美学 |
| **视觉情绪** | 文化底蕴、独特、差异化、高级 |
| **适合年龄段** | 全年龄段（尤其适合"有文化感"的父母） |
| **MVP可实现性** | ⭐⭐ 低 |
| **优点** | 差异化最强、竞品几乎没有这个风格、品牌辨识度极高 |
| **缺点** | AI生成极不稳定、文件偏大、风格统一难度极高、与暖橙配色体系冲突 |
| **情绪一致性** | 取决于配色——国潮≠冷淡，可以用暖色调和 |

**典型参考**：故宫文创、喜茶联名设计、站酷国潮插画

---

### 10. 表情包 / Meme 风

| 维度 | 评价 |
|------|------|
| **特点** | 夸张表情、网络感、易传播、自带社交货币属性 |
| **视觉情绪** | 轻松、搞笑、降低育儿焦虑感 |
| **适合年龄段** | 裂变分享海报、结果页头像 |
| **MVP可实现性** | ⭐⭐⭐ 中 |
| **优点** | 裂变传播最强、年轻父母喜欢、自带"转给老公看"属性 |
| **缺点** | 不适合测评题配图（会分散注意力）、风格难统一、可能显得不够专业 |
| **情绪一致性** | 轻松温暖——但不适合严肃场景 |

**典型参考**：微信表情包、小红书育儿搞笑漫画

## 三、MVP推荐方案

### 推荐：方案A —— 扁平插画为主 + 线条插画为辅（双风格体系）

| 层级 | 风格 | 用途 | 原因 |
|------|------|------|------|
| **主风格（90%）** | 扁平插画 | 28道测评题配图 | 文件小、加载快、统一易控、情绪温暖可控 |
| **辅风格（10%）** | 线条插画 | 13-18岁敏感场景（5道） | 降低"被评判"感、高级不打扰、加载最快 |

**为什么不选其他风格？**
- 水彩/黏土/国潮 → MVP文件大小和开发成本hold不住
- 手绘卡通 → 很好，但风格统一需要同一画师，AI生成批次间差异大
- 极简Icon → 太冷，代入感不够
- 表情包 → 不适合测评场景，会分散注意力

### 配色绑定（与产品色彩体系一致）

```
主色：暖橙 #F5A623（情绪高亮、重点元素）
辅色：暖白 #FFF8F0（背景、柔和过渡）
中性：炭灰 #4A4A4A（线条、文字）
点缀：薄荷绿 #7ED321（正向元素、安全感）
情绪：淡粉 #FFB6C1（温柔场景、孩子元素）
```

### 文件规格

| 参数 | 规格 |
|------|------|
| 尺寸 | 750px × 400px（2:1横幅，适配手机屏） |
| 格式 | WebP（优先）/ PNG |
| 单张大小 | ≤100KB |
| 加载策略 | 首屏3张预加载，其余懒加载 |
| 人物比例 | 人物占画面40-60%，不拥挤 |

## 五、执行建议

### MVP阶段（立即执行）

1. **先不做28张全图** —— MVP只需做3-5张核心场景的插画测试用户反应
2. **用AI生成+人工修图** —— ChatGPT/Claude写prompt → Midjourney/SD出图 → 设计师修图统一风格
3. **首测场景建议**：
   - 超市打滚（0-3岁，情绪冲突最强）
   - 考试藏卷（7-12岁，学业压力典型）
   - 锁门摔手机（13-18岁，青春期冲突）
   - 测评结果页头像（4种画像各一个）

### V1.1阶段（上线后迭代）

1. 根据用户反馈决定是否补全28张
2. 如果数据好，用「扁平插画」补全全部场景
3. 如果用户觉得"太模板"，切「手绘卡通」风格重做

### 风险提醒

- **AI生成人物一致性差**：同一个人物（妈妈/孩子）在28张图里可能长得不一样
- **解决方案**：先定义1-2套人物模板（妈妈形象+孩子形象），所有prompt都绑定这些模板特征
- **人物模板示例**：
  ```
  Mother character: medium-length black hair, wearing casual home clothes 
  (warm orange tone), soft facial features, always shown with gentle or tired expression.
  Child character: round face, short hair, small stature, wearing simple colorful clothes.
  ```
