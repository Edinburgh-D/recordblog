# Obsidian 手机端入门教程 · 「父母指南」项目专用

> 目标：让你在手机上5分钟建好知识库，导入项目全部文档
> 适用：iOS / Android 通用
> 前置条件：手机一台，不需要电脑

---

## 第一步：下载安装 Obsidian（1分钟）

### iOS（iPhone）
1. 打开 **App Store**
2. 搜索 **"Obsidian"**
3. 点击 **获取** → 安装
4. 打开App，点 **Create new vault**（创建新仓库）

### Android
1. 打开 **Google Play**（或应用商店）
2. 搜索 **"Obsidian"**
3. 点击 **安装**
4. 打开App，点 **Create new vault**

---

## 第二步：创建你的第一个知识库（1分钟）

打开Obsidian后，你会看到：

```
Create new vault
  Vault name: [输入名字]
  
  [Create]
```

**操作**：
1. 在 "Vault name" 里输入：`父母指南项目`
2. 点击 **Create**
3. 看到空白界面就是成功了

---

## 第三步：下载项目文件（1分钟）

### 方式A：从GitHub下载（推荐）
1. 手机浏览器打开：`https://github.com/Edinburgh-D/personnal-assets`
2. 找到绿色按钮 **Code** → 点 **Download ZIP**
3. 下载完成后，去手机的 **文件管理器** 里找到这个ZIP
4. 点击ZIP → **解压** → 得到文件夹 `personnal-assets-main`

### 方式B：直接下载我发的文件（更快）
如果你在本对话中收到了文件附件：
1. 长按文件 → **保存到手机**
2. 去文件管理器找到保存的位置

---

## 第四步：把文件导入 Obsidian（2分钟）

### iOS 导入方法

**核心逻辑**：Obsidian的每个vault就是一个文件夹，你把Markdown文件复制进去就行。

1. 打开 **文件App**（系统自带）
2. 找到解压后的文件夹 `personnal-assets-main`
3. 长按文件夹 → **拷贝**
4. 在文件App里找到 **Obsidian** 文件夹（通常在"我的iPhone"或"iCloud Drive"下）
5. 打开 `Obsidian/父母指南项目/`
6. 空白处长按 → **粘贴**
7. **完成！打开Obsidian App，文件已经在了**

> 如果找不到Obsidian文件夹：在文件App里点"浏览"→"我的iPhone"→找到Obsidian→打开你的vault名称。

### Android 导入方法

1. 打开 **文件管理器**（系统自带或第三方）
2. 找到解压后的文件夹 `personnal-assets-main`
3. 长按文件夹 → **复制** 或 **剪切**
4. 导航到手机存储的：`/Android/data/md.obsidian/files/父母指南项目/`
5. 或者更简单：打开Obsidian App → 点左上角 **≡** → **Open folder** → 选择你解压的文件夹
6. **完成！**

> Android路径因品牌而异，如果找不到，用方式B更简单。

### 方式B（Android更简单）：直接在Obsidian里打开

1. 打开Obsidian App
2. 点左上角 **≡**
3. 点 **Open folder as vault**
4. 在文件管理器里找到你解压的 `personnal-assets-main` 文件夹
5. 点 **选择文件夹**
6. Obsidian直接把这个文件夹当作vault打开，所有文件都在里面

## 第六步：Obsidian 高级功能（可选）

### 1. 双链跳转（最实用的功能）
如果文档里有 `[[另一个文件名]]`，点一下就能跳转到那个文件。

比如画像报告里写了 `[[父母指南_题库统一版_v3.1]]`，点一下直接看题库。

### 2. 图谱视图（看文件关系）
1. 点左上角 **≡**
2. 点 **Graph view**
3. 看到所有文件的关系图谱（像蜘蛛网一样）

### 3. 收藏常用文件
1. 打开一个文件
2. 点右上角 **⋮** → **Star**
3. 以后在左侧 **Starred** 里快速打开

### 4. 添加自己的笔记
1. 点右下角 **+** 按钮
2. 输入文件名
3. 写笔记内容（支持Markdown格式）
4. 自动保存

## 常见问题

**Q：文件显示乱码？**
A：确保文件是UTF-8编码的Markdown文件，obsidian原生支持中文。

**Q：图片不显示？**
A：图片需要和Markdown文件在同一个文件夹里，且路径正确。如果图片在GitHub上，需要下载到本地。

**Q：能不能编辑这些文件？**
A：可以。Obsidian支持直接编辑Markdown文件，你的修改会保存在手机上。

**Q：换了手机怎么办？**
A：Obsidian的vault就是一个普通文件夹，你可以用任何方式备份（iCloud、Google Drive、微信文件传输等）。

**Q：需要付费吗？**
A：Obsidian个人使用完全免费。只有同步功能（Obsidian Sync）才收费，不用同步就不花钱。
