# 「父母指南」项目 · 全部核心结论汇总

> 综合来源：内部产品分析 + 手机端AI应用交叉验证
> 项目阶段：预研完成，等待MVP开发启动

---

## 一、产品定位

**小程序名称**：「新手父母持证上岗指南」（简写"父母指南"）
**核心功能**：父母育儿风格测评工具
**一句话卖点**：帮焦虑的父母看懂自己的育儿方式
**目标用户**：0-18岁孩子的家长（焦虑、自我认知需求强）
**技术栈**：微信小程序 + Taro 3.x + Canvas 2D（海报生成）

| 类型 | 组合 | 画像名 | 核心一句话 |

| 纯型 | C | 侦探型父母 | "我想知道你为什么这样" |
| 纯型 | D | 行动派指挥官 | "现在做什么比说什么重要" |
| 双型 | A+C | 共情引导者 | 先懂心再引路（⭐用户指定优先） |
| 双型 | B+D | 秩序建构者 | 先立规矩再行动（⭐用户指定优先） |

| 双型 | C+D | 领航员 | 看透+快改 |
| 三型 | A+B+C | 全能调和者 | 三者都会，精力只够两样 |
| 三型 | A+B+D | 权威共情者 | 有边界的温暖力量 |
| 三型 | A+C+D | 敏锐守护者 | 看见→想通→出手 |
| 三型 | B+C+D | 战略执行者 | 分析→果断 |
| 全型 | A+B+C+D | 全能整合者 | 四维度自由切换 |
| 均势 | 无突出 | 平衡型父母 | 没有招牌动作，但很少做错 |

**判断标准**：
- 纯型：单一维度显著高于其他3个
- 双型：最高和次高维度差值≤2
- 三型：三个维度均高（≥5），第四个低
- 全型：四个维度均高（≥6）
- 均势：四个维度差距均≤1

## 三、题库体系（28道，v3.1）

| 年龄段 | 题目数 | 多孩场景 |
|--------|--------|----------|
| 0-3岁 | 7道 | q025（弟弟拽姐姐辫子） |
| 4-6岁 | 7道 | q026（哥哥锁门不让妹妹进） |
| 7-12岁 | 7道 | q017（抢遥控器）、q027（大宝觉得爸妈偏心） |
| 13-18岁 | 7道 | q028（弟弟翻姐姐日记） |

**四维度分布**：每年龄段7题，每题4选项，各维度（A/B/C/D）在每年龄段出现7次，总计每个维度28次。

**多孩场景**：共5道题标记multi_child=true，覆盖4个年龄段。

**标准化格式**：JSON统一版v3.1，可直接用于开发。每道题含id/age_group/scene_tag/scene/frequency/multi_child/options[]，每个option含letter/dimension/text/result_label/philosophy/message。

## 四、心路历程优化（v1.0）

### 优化前 → 优化后

| 环节 | 原来 | 改后 | 工作量 |
|------|------|------|--------|
| 裂变入口 | "测测你是什么类型的父母" | "这道题我选了A，老公选了D——结果发现居然都对" | 文案 |
| 起始页 | 先填信息（婚姻/有孩/年龄） | **直接开始测评**，全程不阻断 | 删页面 |
| 测评中 | "每题90秒" | 进度条+"选你第一反应，没有标准答案" | 删文案 |
| 结果页 | "你是温室园丁" | "面对孩子崩溃，你倾向于先接住情绪"+雷达图 | 文案+UI |
| 转化 | "分享解锁完整报告" | 基础报告全给，生成海报自发分享 | 产品逻辑 |
| 复访 | 无 | 添加小程序/关注公众号/3天后推送 | 文案埋点 |

### 四家AI交叉验证的高共识

4. 双人/家庭对比 ✅ 共识
5. 社交货币型文案 ✅ 共识

## 五、自主支持维度融入

**结论：MVP不新增维度，V2.0再考虑。**

- **方案A（新增E维度）**：学术严谨，但工作量爆炸（每题5选项、画像16→25种、全部重测）→ MVP不建议
- **方案B（画像加分项）**：零代码改动，1-2天完成 → **推荐MVP执行**

**方案B具体执行**：
1. 标记现有28题中含自主支持的10个选项（6个直接+4个暗含）
2. 结果页增加"赋权指数"（0-10分，根据选了几个赋权选项）
3. 深度报告增加"自主支持专项建议"（按年龄段推送）
4. 文案永远正向表达，不说"好坏"只说"方向"

## 六、插画风格测试

| 风格 | 测试结果 | MVP推荐 |
|------|---------|---------|
| 手绘卡通 | 可爱，0-6岁适配最好 | 备选 |
| 扁平插画 | 小程序UI感强但情绪表达弱 | 不推荐（妈妈表情微笑=冷漠旁观） |
| 水彩风 | 治愈感强但场景有偏差 | 情绪题专用（远期） |
| **绘本风** | **最温暖，故事感最强，情绪准确** | **⭐首推** |
| 国潮风 | 有设计感但不像日常场景 | 不推荐 |
| ChatGPT/DALL-E | 理解细腻情绪（蹲下伸手+心疼表情） | 手机上最佳生图工具 |
| Gemini | 色块风太抽象，情绪表达弱 | 不推荐 |

**MVP执行建议**：不做28张全图，只做5张核心场景（起始页+结果页+3个年龄段首题）。人物锁定：麻花辫妈妈+黄毛衣+心疼表情。

### 结果页文案示例（去Judgemental化）

```
"你的育儿风格偏向'秩序建构者'——
这和传统中国文化中的'严父慈母'观念有相似之处。
你不是在'控制'孩子，你是在用'规则'为孩子建立安全感。

研究表明（Baumrind, 1966），高规则的环境能培养孩子的
自律能力和责任感。

同时，小贴士：
规则之外，孩子也需要知道'被接纳'的感觉。
试着在规则执行前，先说一句'我理解你'。"
```

## 八、AI应用能力画像

| AI | 擅长 | 最佳任务 | 短板 |
|----|------|----------|------|
| **ChatGPT** | 逻辑严密、找矛盾、执行方案 | 题库审校、流程优化 | 偶尔偏理想化 |
| **Perplexity** | 数据背书、学术引用、竞品参考 | 竞品分析、理论框架 | 服务端不稳定 |
| **Grok** | 犀利吐槽、接地气、用户直觉 | 选项真实感测试、找茬 | 回复可能不完整 |
| **Claude** | 深层心理学、体系扫描、长文报告 | 体系漏洞扫描、深度预演 | 服务端故障（当前不可用） |
| **Gemini** | 文案策略、UI/UX、微信生态联动 | 裂变文案、界面设计 | 操作路径不稳定 |
| **DeepSeek** | 中文语感、产品设计、IP化创意 | 中文润色、社交锚点文案 | 不能生图 |

**核心原则**：任何任务至少用2个AI交叉验证；ChatGPT+Grok是题库打磨黄金组合。
